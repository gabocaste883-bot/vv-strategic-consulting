# V&V Strategic Consulting Landing Page

Static bilingual landing page ready to upload to GitHub Pages, Netlify, Vercel, or any hosting service.

## Files

- `index.html` – main landing page
- `assets/css/styles.css` – design and responsive layout
- `assets/js/main.js` – language switcher and animations
- `assets/images/` – hero image, portrait and favicon

## How to use on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Select **Deploy from branch**.
5. Choose branch `main` and folder `/root`.
6. Save and wait for GitHub to publish the site.
